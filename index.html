Grid

Built on flexbox, this support components when placing it on a page. The grid is flexible and resize all your component to a reasonable size making it look consistent and functional on all screen size.

By default, desktop screens has a maximum of 12 columns, tablet for 8 columns, and phones for 4 columns. You can change the value of the maximum columns on each screen by changing the column variable on bae.

.cell.cell--12
.cell.cell--8
.cell.cell--4

By just using cell, this will stack all other cells and place them in grid neatly and evenly. Automatically size all cells on a row.

cell

cell

You can set your desired cell size on each screen, making it flexible and suits your preference.

Note that:
  • p - phone
  • t - tablet
  • d - desktop
.cell.cell--p4.cell--t8.cell-d12
.cell.cell--p4.cell--t2.cell--d6
.cell.cell--p4.cell--t6.cell--d6

Hiding a certain cell on different screen size is easy, by applying is-<screen size>-hidden class, it will hide everything inside of that cell.

.cell.cell--12.is-tablet-hidden
.cell.cell--12.is-phone-hidden
.cell.cell--12.is-desktop-hidden

NOTE: Hiding can apply on other component as well but it is preferred to use it on a cell

Cell placement are easy. You can place a cells anywhere inside a grid

grid.grid--center > cell.cell--6.cell--middle

.grid.grid--center > .cell.cell--6.cell--middle placing cell in the middle of a grid.

grid.grid--right > cell.cell--6.cell--bottom

.grid.grid--right > .cell.cell--6.cell--bottom placing cell in the bottom right of a grid.

To check all grid modifiers and decendant of the bae, check out its documentation.
